点击
# 使用
## 桌面版
- 打开FastGithub.UI.exe即可
## 服务版
### 以Windows服务的方式运行
- 在文件夹的地址栏键入cmd.exe
- 执行以下命令以启动服务
```
fastgithub.exe start
```
- 如果需要关闭使用以下命令
```
fastgithub.exe stop
```
### 另一种方法
#### 启动
直接双击运行fastgithub.exe即可，出现的窗口不可以关闭。
#### 关闭
关闭窗口即可。
